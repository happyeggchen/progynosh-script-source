installer
=========
    A installer mod for progynosh
      Include:
        2 functions
          {install,uninstall}
      Usage:
        install $argv[1] >>> to install script itself to /usr/bin
          $argv[1] : the filename in /usr/bin
        -----------------------------------------------
        uninstall $argv[1] >>> to remove script itself from /usr/bin
          $argv[1] : the filename in /usr/bin
